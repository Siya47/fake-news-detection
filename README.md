# Fake News Detection System

This project detects whether a news article is **Fake or Real** using Natural Language Processing (NLP) and Machine Learning techniques.  
The model analyzes textual patterns in news articles and classifies them using a Logistic Regression model.

---

## Project Overview

Fake news has become a major problem in digital media. This project builds a machine learning model that can automatically classify news articles as **Fake** or **True** based on their textual content.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- TF-IDF Vectorization
- Matplotlib
- Seaborn

---

## Dataset

The dataset contains two types of news articles:

- **Fake.csv** → Fake news articles
- **True.csv** → Real news articles

Each article contains text data used to train the model.

---

## Project Workflow

1. Import libraries and load datasets
2. Label fake and true news
3. Merge datasets
4. Perform text preprocessing
5. Apply TF-IDF vectorization
6. Train machine learning model
7. Evaluate model performance

---

## Machine Learning Model

The model used in this project:

- **Logistic Regression**

This algorithm is commonly used for text classification problems.

---

## Results

The model predicts whether a news article is **Fake** or **True** based on its content.  
Evaluation is performed using accuracy score and classification report.
